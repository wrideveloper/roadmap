# Cyber Security

Selamat datang di miniclass Cyber Security. Pada fase awal ini, roadmap difokuskan pada jalur **Penetration Testing Fundamental** untuk membantu peserta memahami alur kerja pentest dari recon sampai reporting secara aman dan terstruktur.

## Jalur Belajar

- [Penetration Testing Roadmap](/miniclass/cyber/penetration-testing/)

## Cakupan Materi

Miniclass ini saat ini mencakup:

- Etika, scope, dan metodologi penetration testing
- Passive dan active reconnaissance
- Enumeration dan attack surface mapping
- Validasi kerentanan dan eksploitasi dasar
- Web application pentesting dasar
- Credential attacks dan post-exploitation dasar
- Evidence handling, risk rating, dan report writing
- Mini capstone untuk mensimulasikan alur pentest end-to-end

## Catatan

Struktur materi di bawah merupakan scaffolding awal berdasarkan syllabus penetration testing. Setiap halaman sudah disiapkan agar mentor dapat mengembangkan detail teori, demo, praktikum, dan lampiran tugas secara bertahap.
