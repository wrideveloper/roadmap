## 🌐 Apa itu Web Programming?

Web Programming atau **Pemrograman Web** merupakan istilah yang erat kaitannya dengan internet dan website. Pemrograman web adalah proses pembuatan website untuk keperluan internet. Orang banyak mengenal web dengan istilah **WWW** atau **World Wide Web**.

:::tip 💡 Tahukah Kamu?
World Wide Web adalah halaman-halaman website yang terkoneksi satu dengan lainnya atau disebut juga dengan **hyperlink** yang membentuk samudera informasi yang berjalan dengan protokol **HyperText Transfer Protocol (HTTP)**.
:::

## 💻 Bahasa Pemrograman Web

Dalam pemrograman web, terdapat berbagai bahasa pemrograman yang digunakan untuk membuat aplikasi web yang berisi statement-statement, perintah, atau fungsi agar aplikasi berfungsi sesuai dengan yang didesain.

### 🔧 Teknologi Fundamental

#### 1. **HyperText Markup Language (HTML)**

:::info 📝 Catatan
HTML sebenarnya bukan sebuah bahasa pemrograman, melainkan **markup language** atau bahasa penandaan yang terdiri dari kumpulan tag untuk mendeskripsikan struktur konten halaman web.
:::

#### 2. **Cascading Style Sheet (CSS)**

CSS merupakan bahasa stylesheet yang digunakan untuk mengatur tampilan suatu dokumen yang ditulis dengan HTML. CSS juga memiliki CSS framework dan digunakan untuk menambah desain-desain tertentu pada halaman web agar tampilan menarik.

#### 3. **JavaScript**

JavaScript adalah bahasa scripting yang berjalan pada sisi **client**. Pemrosesan script dilakukan pada komputer user. JavaScript digunakan untuk membuat animasi dan bentuk interaktif lain pada halaman web.

#### 4. **PHP**

PHP (Hypertext Preprocessor) adalah bahasa pemrograman open-source yang umumnya digunakan untuk membangun aplikasi web dinamis dan interaktif. PHP dapat dijalankan pada server web dan dikombinasikan dengan HTML, CSS, dan JavaScript.

#### 5. **Structured Query Language (SQL)**

SQL merupakan domain-specific language yang digunakan untuk mengolah data dalam **Relational Database Management System (RDBMS)**. Aplikasi RDBMS yang populer adalah MySQL.
