# ⚡ Backend vs Frontend

## 🎯 Perbedaan Utama

Frontend dan backend adalah dua aspek penting dari aplikasi apa pun:

- **Frontend** = Apa yang pengguna **lihat** dan **gunakan**
- **Backend** = Apa yang terjadi di **balik layar**

## 🎨 Frontend Development

:::tip 💡 Apa itu Frontend?
Frontend adalah bagian dari situs web yang **berinteraksi langsung dengan pengguna**. Ini termasuk desain, menu, teks, gambar, video, dan tata letak keseluruhan.
:::

### Teknologi Frontend:
- **HTML** - Struktur konten
- **CSS** - Styling dan tampilan
- **JavaScript** - Interaktivitas dan dinamis

### Fokus Utama:
✅ **Responsivitas** - Tampilan baik di berbagai ukuran layar  
✅ **Kinerja** - Loading cepat dan smooth  
✅ **User Experience** - Mudah digunakan dan intuitif

## ⚙️ Backend Development

:::tip 💡 Apa itu Backend?
Backend adalah **sisi server** dari situs web. Backend mengelola data dan memastikan semua hal di frontend berfungsi dengan baik.
:::

### Karakteristik Backend:
- 🔒 Pengguna **tidak melihat** atau berinteraksi langsung
- 🔄 Mengelola **logika bisnis** aplikasi
- 💾 Mengelola **database** dan penyimpanan data
- 🔐 Menangani **autentikasi** dan **keamanan**

### Aktivitas Backend Developer:
- Menulis **API** (Application Programming Interface)
- Membuat **library** dan **framework**
- Bekerja dengan **database**
- Mengelola **server** dan **infrastruktur**
- Implementasi **business logic**
